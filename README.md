# Odin: Rock Paper Scissors

A simple, console-based implementation of Rock Paper Scissors created as part of **The Odin Project** Foundations curriculum.

## How to Play
1. Clone the repository or download the files.
2. Open `index.html` in any web browser.
3. Open the **Developer Tools Console** (Right-click > Inspect > Console, or press F12).
4. The game will play 5 rounds and announce the final winner!

## Lessons Learned
* Using JavaScript functions to return random values.
* Managing game state through variables (`humanScore`, `computerScore`).
* Implementing loop logic for multi-round gameplay.
* Handling user input and normalizing strings with `.toLowerCase()`.
